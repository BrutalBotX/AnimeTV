# AnimeTV

AnimeTV is a cross-platform anime streaming application for Android TV, Google TV, Fire TV, Android smartphones, Windows, Linux, and macOS.

It is optimized for TV interfaces with full TV remote navigation support, no ads, no data collection, and no analytics.

## Features

- Browse and stream anime from multiple sources
- TV-friendly interface with remote control navigation
- Skip intro, playback speed control, subtitle support
- Watchlist, history, and episode tracking
- Customizable themes and wallpapers
- Profile support with custom avatars
